E-Voting System

This repository contains documentation and design artifacts related to an E-Voting System project.

Note: Due to technical issues with my PC, the original soft copy of the project was lost. However, the hard-copy documentation was preserved, and screenshots were captured from the available hard-copy materials to document the project. A few original soft-copy files that were successfully recovered, including design documents such as DFDs, ER diagrams, and related artifacts, have also been uploaded to this repository.

The uploaded materials are intended to preserve and showcase the project's design, analysis, and development work.
